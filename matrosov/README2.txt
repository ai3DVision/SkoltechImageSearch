Description of files:

word_oxc1_hesaff_sift_16M_1M/
-----------------------------
Directory containing word ids (for a vocabulary of 1M words) and geometry for
the 5K dataset. The geometry files have the following layout:
1. The first two lines can be ignored.
2. Every subsequent line consists of [WORD_ID] [X] [Y] [A] [B] [C]
X and Y is the feature centroid and A, B, C define the parameters of
the ellipse in the following equation (as per http://www.robots.ox.ac.uk/~vgg/research/affine/detectors.html):
A*(x-X)^2 + 2*B*(x-X)*(y-Y) + C*(y-Y)^2 = 1
